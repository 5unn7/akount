# Analytics

Visual financial reporting and analysis dashboard with cash flow tracking, P&L statements, balance sheets, and AI-powered insights. Filter by entity and time period to understand financial performance and trends.

## Key Features
- Dashboard overview with key metrics and trends
- Cash flow analysis showing money in vs. out over time
- Category breakdown charts
- P&L (income statement) generation
- Balance sheet snapshots
- AI-generated financial insights and recommendations
- Time period selector (this month, last quarter, YTD, custom ranges)
- Entity filter for specific or consolidated views
- Export to PDF and CSV
- Saved report configurations

## Tab Navigation
- **Dashboard** (default)
- **Cash Flow**
- **P&L**
- **Balance Sheet**
- **Insights**

## Components
- Metric cards with trend indicators
- Line charts for cash flow
- Category breakdown visualizations
- Structured P&L and balance sheet views
- Insights panel
- Export buttons
- Report filter controls
