# Planning

Financial planning tool for setting budgets and goals with progress tracking. Create budgets by category and entity, define savings and debt paydown goals with target dates, and use financial calculators for scenario planning.

## Key Features
- Budget creation by category, entity, or both with time periods
- Budget variance analysis (actual vs. budgeted) with progress bars
- Financial goal setting (savings, debt paydown) with target dates
- Goal progress tracking with suggested monthly contributions
- Financial calculators (savings, debt payoff, budget allocation, retirement)
- Alerts and warnings when budgets exceeded or goals off-track
- Flexible grouping by category, entity, or combination
- Visual progress bars and charts

## Tab Navigation
- **Budgets** (default)
- **Goals**
- **Calculators**

## Components
- Budget list with inline creation
- Budget rows with progress bars and alerts
- Goal list with progress visualization
- Goal rows with target dates and suggestions
- Four interactive financial calculators
- Color-coded alert system (green/yellow/red)
