# Foundations

## Purpose

Foundations contain the design primitives that form the base of the entire design system: colors, typography, spacing, animation, and design tokens. These are the building blocks that all components and patterns build upon.

## Files in This Category

- **`philosophy.md`** - Core design principles, mental models, and emotional outcomes
- **`colors.md`** - Color system with semantic meanings for finance, AI, and states
- **`typography.md`** - Font families, type scale, and usage rules
- **`spacing-layout.md`** - Grid system, spacing scale, and responsive guidelines
- **`motion.md`** - Animation principles, easing functions, and timing
- **`tokens/README.md`** - Token documentation and CSS variable usage guide
- **`tokens/akount.tokens.json`** - Design token source of truth
- **`tokens/css-variables.css`** - Unified CSS variables (consolidated from 3 files)

## When to Use

Reference these files when:
- Creating or updating design tokens
- Establishing consistent color usage across components
- Setting typography standards
- Defining spacing and layout structure
- Planning animations and micro-interactions
