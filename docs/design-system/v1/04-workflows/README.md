# Workflows

## Purpose

Workflows describe user journeys and multi-step interactions that guide users through complex tasks. They show the sequence of screens and patterns a user encounters when completing a specific workflow.

## Files in This Category

- **`onboarding-bank.md`** - Bank account connection, statement import, account matching workflow
- **`onboarding-accountant.md`** - Accountant invitation, permissions setup, collaboration activation
- **`entity-selection.md`** - Multi-entity awareness, entity switching, context persistence
- **`accountant-collaboration.md`** - Real-time collaboration, document requests, approval workflows
- **`rules-automation.md`** - Rule creation, condition matching, bulk application, scheduling

## When to Use

Reference these files when:
- Understanding multi-step user journeys
- Planning user flows for new features
- Coordinating across multiple screens
- Documenting complex interactions
